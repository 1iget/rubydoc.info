Learn More About YARD

What is YARD?

YARD is a documentation generation tool for the Ruby programming language. It enables the user to generate consistent, usable documentation that can be exported to a number of formats very easily, and also supports extending for custom Ruby constructs such as custom class level definitions. What's this mean? It means YARD is a great tool to document applications that make use of DSLs.

YARD uses @tag style definitions (like in Java, Objective-C, Python and other languages) to add rich meta-data that can either be formatted in your HTML docs or accessed programmatically for plenty of use cases.

You can find out much more about YARD's features in the README.

Who Uses YARD?

DataMapper, Haml, ThoughBot, rdoc.info to name a few. Maybe even Rails?

A full list can be found here.

How to Use YARD

You can use YARD to do many things. The most common is to generate documentation. Simply use:

$ yardoc lib/**/*.rb

HTML documentation will now be generated to your doc/ directory.

You can also use YARD to do auditing on documentation by writing extensions. An overview of the API can be found here.
